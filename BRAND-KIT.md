# MARGIN — Brand Kit

Eight PNGs on the Desktop, rendered from `_studio/` (run `node build.js && node render.js`).
Regenerate any single asset with `node render.js margin-<name>`.

| | |

## Palette

| Token | Hex | Use |
| --- | --- | --- |
| Black | `#0a0d0a` | background |
| Panel | `#121712` | cards |
